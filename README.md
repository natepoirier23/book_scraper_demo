# Book Scraper Demo

A clean, minimal Python scraper for [BooksToScrape.com](https://books.toscrape.com).

This project demonstrates:
- Requests-based fetching
- HTML parsing with BeautifulSoup
- Pagination handling
- CSV and JSON export
- Modular scraper architecture

This scraper **does not use Playwright** and is intended as a simple demo project for portfolios.

## Usage

```bash
python main.py
